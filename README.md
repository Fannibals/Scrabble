## How to import the project
JDK required: 1.8
### To Eclipse  
import existing maven projects --> open the directory you downloaded --> finish
### To IntelliJ IDEA  
open directly --> open project structure --> set jdk to 1.8 and module as 8(Lambda)  
## How to package  
Maven required: 3.5 +  
Command: ` mvn package `
